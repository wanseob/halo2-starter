# Halo2 Starter Kit

## Pre-requisites
1. Install rust

## Instructions

1. Go to `src/chip/field_config.rs` and configure the chip.
2. Go to `src/chip/numeric_instructions.rs` and define the instruction and then implement the chip.
3. Go to `src/circuit.rs` and implement the circuit.


## Commands
1. Test
    ```
    cargo test
    ```
